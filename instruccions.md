# PROYECTO: TRADUCTOR DE AUDIO DEL SISTEMA EN TIEMPO REAL CON SUBTÍTULOS FLOTANTES

## OBJETIVO

Crear una aplicación para Windows 11 que:

1. Escuche TODO el audio que sale por los altavoces del sistema.
2. Detecte automáticamente cuando el idioma hablado es inglés.
3. Transcriba el audio en tiempo real usando GPU NVIDIA RTX 5060TI 16GB VRAM.
4. Traduzca automáticamente al español.
5. Muestre subtítulos flotantes sobre cualquier ventana.
6. Mantenga siempre la ventana visible encima de:
   - YouTube
   - Netflix
   - VLC
   - Discord
   - Zoom
   - Juegos
   - Navegadores

7. La aplicación debe funcionar completamente local.
8. Haya una pequeña ventana donde se pueda seleccionar y guarda el dispositivo de audio
